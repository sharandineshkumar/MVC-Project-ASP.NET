# MVC Complaint System — Master Context

## Project Overview
ASP.NET Core MVC Complaint Management System
- Citizens register, login, submit complaints
- Admins view all complaints, edit status, delete
- Built with .NET 10, EF Core, SQL Server, Bootstrap 5
- Cookie Authentication, SHA256 password hashing

## Database
- Server: localhost\SQLEXPRESS
- Database: ComplaintSystemDB
- Tables: Users, Complaints

## User Roles
- Citizen → register, login, submit complaints,
  view OWN complaints only, edit OWN complaints
- Admin → view ALL complaints, edit status, delete
- Admin role set manually in SSMS:
  UPDATE Users SET Role = 'Admin' WHERE Email = 'youremail@gmail.com'
